# 📊 Data Analytics Internship Portfolio

A four-week hands-on Data Analytics Internship focused on developing practical skills in **Excel, Python, SQL, Machine Learning, and Power BI** through weekly assignments and an end-to-end capstone project.

This repository documents my complete learning journey—from cleaning raw datasets in Excel to building predictive models and interactive Power BI dashboards.

---

## 🎯 Internship Objectives

- Learn industry-standard data cleaning techniques
- Perform exploratory data analysis (EDA)
- Analyze business data using SQL
- Build interactive dashboards
- Apply machine learning for sales prediction
- Present business insights through reports and visualizations
